Программа-секундомер, написанная в качестве тестового задания для Napoleon IT/

Доработки:
Можно сохранять значения секундомера в выбранный текстовый файл, дописав комментарий к сохраняемому результату.

Запуск программы для Windows возможен из папки Desktop_run(в папке подключены все необходимые библиотеки)
